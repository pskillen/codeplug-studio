# ChannelBehaviourExportOverrides

## Purpose

Optional per-build overrides for library channel behavioural defaults on the Export panel. Each field uses a switch — when enabled, the segment value is persisted on `FormatBuild.exportSettings` and wins over library + per-channel cascade at export.
